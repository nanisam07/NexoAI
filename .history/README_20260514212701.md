# Nexora AI 🚀

An AI-powered modern Learning Management System (LMS) built using Next.js, Firebase, Tailwind CSS, and Framer Motion.

Nexora AI delivers a futuristic learning experience with:

* OTP & Google Authentication
* AI-powered course ecosystem
* Dynamic student dashboard
* Firestore backend integration
* Real-time enrollments
* Responsive modern UI
* Protected routes
* Interactive course modules
* Payment flow simulation
* Student-specific course tracking

# 🔐 Authentication Flow

## OTP Authentication

1. User enters phone number
2. Firebase sends OTP
3. OTP verification occurs
4. User data stored in Firestore
5. User redirected to dashboard

## Google Authentication

1. User clicks Google Login
2. Firebase popup authentication
3. User profile stored in Firestore
4. User redirected to dashboard
